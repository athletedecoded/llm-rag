# Rusty RAG

LLM w/ RAG from scratch using Rust & Ollama

**Setup**

```
# Install ollama
$ make ollama

# Get model and tokenizer files
$ make models
```

**Seed RAG Database**

```
$ cargo run --bin seed
```

**Scan RAG Database**

```
$ cargo run --bin scan
```

**Launch**

```
$ cargo run
```
